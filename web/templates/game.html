Animal invasion!

This is the first larger-scale 2D game that I've created, written in Python programing language, using the Arcade game developer module (Check it out on github). This game turned out to be quite large, but I believe it was worth the invested energy because it has become a very enjoyable game! In this game you are the main character, a businessman, who tries to collect bitcoins throughout the challenges of the game, guarded by animals and other dangerous creatures. These entities inflict damage upon contact, so it's better to avoid them to prevent any fatal consequences. But don't forget to collect the bitcoins that these entities guard, because if you successfully gather all 20 bitcoins, I'll reward you with a heartfelt thank you! Below, you'll find images of the game and instructions for launching. I wish you a pleasant gaming experience!

It's possible that your computer might hesitate to open the file due to potential risks associated with .exe files. I've checked my game using VirusTotal, which scanned it with 63 antivirus engines. The final result was that more than 85% of the antivirus engines considered the game safe, while the remaining ones flagged it as risky or potentially containing a virus. This is because the game has a .exe extension, which is often considered risky by these antivirus engines. However, you can view the results of the virus scan by clicking here. The game is open-source, so you can also inspect and download the game's code by clicking here. So, there's no reason to worry.

Please click here for the guide on how to launch the Game!
  1. After the download is complete, extract the files.
  2. Afterward, open the extracted folder.
  3. Open the folder named 'dist'.
  4. Open the next folder as well.
  5. To start the game, open the ANIMAL_INVASION.exe file.
  6. I wish you the best of luck with your gameplay!
